Install notes

  • Extract the latest php-test.zip file to the htdocs directory.
  • The main thing you need to set up is the database installation. This is stored in the php/database/ directory, and is a copy of the mysql employees sample database.
  • To install the database run:
cd path/to/htdocs/php/database
mysql -u root -h 127.0.0.1 < employee.sql
  • This will set up over 300k employee records & associated other tables.
  • Copy the whole php-test directory to the desktop & call it 'Test Backup'.
  • Back in htdocs/php-tests, delete the 'database' directory (be careful you are not deleting it from the directory on your desktop.

Preparing the machine for the test

  • Open the code directory.

  • Open sublime text - ensure php files are set to open in sublime.

  • Open the web browser at http://localhost/php-test/Test Instructions.html

  • Open another tab at http://localhost/php-test/test1.php

  • Open another tab at phpmyadmin on the local server. Set the database to be the employee database.

After the test

  • Zip all directories.

  • Name using the following format: Php Tests- <date> - <interviewee name>.zip

  • Upload the resulting file to an appropriate recruitment folder on Google Drive.

  • Once upload is successful, copy the zip file off onto a backup USB drive.

  • Delete all contents from htdocs/php-tests, and re-copy them from inside the backup folder on your desktop.
